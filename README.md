
## 🌱 种植系统使用指南

### 种植流程改进

#### 1. 点击"种植"按钮
- 显示**所有可用种子**的选择窗口
- 🟢 **绿色文字**："点击种植" - 表示有库存，可直接种植
- 🔵 **蓝色文字**："点击购买种植" - 表示需要购买后种植

#### 2. 种子信息显示
每个种子显示：
- **种子名称**
- **价格/库存**：有库存显示数量，无库存显示价格
- **生长时间**：完全成熟需要的时间
- **操作提示**：种植或购买种植

#### 3. 购买种植功能
- 自动购买种子并立即种植
- 消耗对应金币
- 种植成功后关闭窗口

- 天气系统影响作物生长

## ☁️ **云端存档功能**

### 🚀 **GitHub Gist存储**
- 完全免费的云端存档
- 数据存储在你的GitHub账号中
- 支持多设备同步
- 4个独立存档槽位

### 🔧 **设置方法**
1. **创建GitHub Token**
   - 访问 [GitHub Settings](https://github.com/settings/tokens)
   - 创建新Token，只需要 `gist` 权限
   - 复制生成的Token

2. **在游戏中设置**
   - 打开调试面板（点击右下角蓝色圆球）
   - 点击"☁️ 云端存档设置"
   - 输入Token并测试连接
   - 保存设置完成配置

3. **自动同步**
   - 游戏数据自动同步到云端
   - 启动时加载最新数据
   - 支持离线游戏
